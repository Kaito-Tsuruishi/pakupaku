# 設計原則:
# - 「言っていないこと」は補わない、「意味が変わる語」は消さない
# - ここに載せるのは、純粋に発話の埋め草で、除去しても意味が変わらない語のみ
# - 副詞・接続詞として本来の意味で使われうる語は含めない
#   (例: 「ちょっと」「だから」「それで」は除外)
# - 「すみません」「ごめんなさい」「失礼」は呼びかけ・謝罪として意味があるので除外
#

# 注意:
# - 「あの」「その」「この」は連体詞 (「あの本」「この件」) としても使うため辞書に入れない。
#   感動詞として Sudachi が判定したケースのみ除去する (filler.py の品詞ベース判定で対応)。
# - 「うん」「ええ」「はい」は応答語 (Yes) なので除去しない。
# - 「なんか」は副助詞・副詞として「何か」の意味で使うため除去しない。
# - 「ちょっと」は副詞「少し」「軽く」の意味で使うため除去しない。
# - 「それで」「だから」「ですから」は接続詞として意味があるため除去しない。
